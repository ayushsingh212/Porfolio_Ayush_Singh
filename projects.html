FARMINGOO

FARMINGOO is an innovative platform bridging the gap between the primary and tertiary sectors through modern technology.

Technologies Used

The frontend of the platform is built using HTML, CSS, and JavaScript.

The backend development leverages Spring Boot along with SQL (Database Management System).

Key Features

The platform includes the following functionalities:

  1. Vehicle Booking System
  2. Order Management System
  3. AI-Powered ChatBot
  4. Search and Filter Tools
  5. User Registration and Login
  6. Weather Support Integration
